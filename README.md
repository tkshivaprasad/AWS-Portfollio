# Shivaprasad Online Resume Website

This is a simple personal online resume website built using HTML and CSS.

## Files
- `index.html`
- `style.css`

## Before uploading to GitHub
Update these placeholders in `index.html`:
- Email
- Phone number
- GitHub username

## How to upload to GitHub
1. Create a new repository in GitHub
2. Upload all files from this folder
3. Commit the files
4. Go to **Settings > Pages**
5. Select branch: `main`
6. Select folder: `/root`
7. Save

Your website will be live through GitHub Pages.
